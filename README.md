# React.js Learning Journey 🚀

This repository tracks a complete React.js learning path from zero to building practical projects.

## 1) Fundamentals
- What React is and why component-based UI matters
- JSX, rendering, and virtual DOM basics
- Props vs state
- Event handling and conditional rendering
- Lists, keys, and basic styling patterns

## 2) Components
- Functional components and composition
- Reusable UI patterns
- Controlled vs uncontrolled form components
- Parent-child communication and lifting state up

## 3) Hooks
- `useState` for local state
- `useEffect` for side effects and lifecycle behavior
- `useRef` for mutable values and DOM access
- `useMemo` / `useCallback` for optimization
- Custom hooks for reusable logic

## 4) Routing
- Client-side navigation with React Router
- Route parameters and nested routes
- Layout routes and shared UI
- Handling 404 pages and protected routes

## 5) Real-World Projects
- Project 1: To-do app (CRUD + local persistence)
- Project 2: Notes or blog app (forms + routing)
- Project 3: API dashboard (fetching, loading, and error states)
- Project 4: Auth-based app (protected routes + state flow)

## Suggested Workflow
1. Learn one topic
2. Build a mini feature
3. Refactor using best practices
4. Ship a small project
5. Repeat with higher complexity

---

Keep this repo updated with notes, code samples, and project milestones as your React skills grow.
